# Belgian e-invoicing context

Belgium requires structured electronic invoicing for nearly all transactions between Belgian enterprises liable to VAT from 1 January 2026. Scope and exceptions must be checked against current official guidance.

Wexio treats invoicing and Peppol as separate layers:

- internal invoice preparation and workflow can exist inside Wexio;
- external Peppol transmission requires a configured provider, operational controls, and recorded transmission proof.

External Peppol provider transmission is not yet provider-proven. This documentation does not claim that Wexio currently satisfies every legal, technical, archiving, or interoperability obligation for every business.
