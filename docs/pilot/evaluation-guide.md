# How to evaluate Wexio during a pilot

Updated: 2026-07-04

This guide helps a pilot company evaluate Wexio in a structured way.

## 1. Start from real work

Do not evaluate Wexio only by asking whether it has many modules. Start from real daily work:

- How do users find client information today?
- Where are documents stored?
- How are tasks followed?
- How are invoices prepared?
- How is staff or time information tracked?
- What is currently repeated manually?

## 2. Choose a narrow first scope

A good first pilot scope is small enough to test quickly and concrete enough to show value.

Examples:

- documents plus client follow-up;
- time tracking plus HR basics;
- products plus invoicing preparation;
- projects plus workflow visibility;
- support requests plus document sharing.

## 3. Define users and roles

List the first users and their expected responsibilities.

Example:

| User type | What to check |
|---|---|
| Owner | Can they understand the workspace value? |
| Admin | Can they manage users and modules? |
| Employee | Can they work without confusion? |
| Worker | Do they receive a focused screen? |
| Client | Do they only see what is intended? |

## 4. Test the workspace experience

Ask users to complete realistic actions:

- open the workspace;
- find the right module;
- use a document;
- follow a task or workflow;
- check a widget;
- switch between business areas;
- understand what is locked or unavailable.

## 5. Measure value

Useful pilot metrics include:

- time saved on a repeated task;
- fewer places to search;
- fewer manual follow-ups;
- clearer ownership;
- fewer duplicate records;
- better visibility for the owner or manager;
- better confidence in access boundaries.

## 6. Decide next steps

After the pilot, decide whether to:

- continue with the same scope;
- add a module;
- improve onboarding;
- adjust roles;
- prepare provider configuration;
- stop if value is not proven.

## Current boundary

Wexio is ready for selected controlled pilot demonstrations. A pilot is not the same as unrestricted production rollout.
