# Controlled pilot status

## Public decision

Wexio is ready for selected controlled pilot demonstrations. This does not authorize unrestricted public production rollout.

## Publicly safe evidence

Controlled verification has covered:

- two-way isolation between separate non-customer proof tenants;
- tenant and user-bound personal workspaces;
- module and widget boundaries;
- private documents and explicit same-tenant sharing;
- Admin versus SuperAdmin destructive-action boundaries;
- build, type, security, database-policy, runtime, and browser gates.

No customer tenant was used for the published proof summaries.

## Limitations

- Pilot scope must be selected and documented.
- Some advanced workflows still require role/module-specific validation.
- Legal templates require professional review.
- Mollie is not configured.
- External Peppol provider transmission is not yet provider-proven.

Detailed internal logs, identifiers, infrastructure information, and attack-surface analysis are intentionally not public.
